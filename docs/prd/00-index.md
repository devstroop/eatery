## Summary

Eatery is a family of four applications forming a complete restaurant management system:

| App | Purpose | Users | Devices |
|-----|---------|-------|---------|
| **Admin** | Full management, config, POS, reports | Owner, manager | Desktop, tablet, phone |
| **Waiter** | Order taking, table management | Waitstaff | Phone, tablet |
| **Kitchen (KDS)** | Real-time order feed, KOT display | Chefs | Wall tablet, desktop |
| **Display** | Customer-facing order status | Customers (read-only) | Kiosk tablet |

Development is organized into 9 phases (P0–P8), starting with core extraction and ending with cloud sync.
